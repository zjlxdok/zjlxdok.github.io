CentOS 安装 Python 3.9 教程

发表于2022-08-26|更新于2022-08-27|教程环境部署
|字数总计:371|阅读时长:1分钟|阅读量:

使用编译安装的方法，不影响原先的 Python 环境

安装依赖

安装所需要的依赖，否则会出现奇奇怪怪的错误

下载源码

服务器上从 Python 官网下载源码包，解压并进入文件夹中

需要事先执行 yum install -y wget（已安装可以忽略）

如果你想安装其他版本，只需要对于更改掉下面的文件名即可

编译安装

--prefix：后跟安装目录

--enable-optimizations：优化性能，但是会极大增加编译时间

如果遇到错误终止，可以执行下面的命令重试

添加环境变量

请对应修改 /usr/local/python3 为 --prefix 指定的目录

安装目录后记得加上 bin，此文件夹下存放着可执行文件

此命令不会立即生效，需要先重启一次，并在下一次重启仍能继续生效（永久有效）

想要立即生效可在命令行中执行 export PATH=$PATH:/usr/local/python3/bin（临时有效，记得更改路径）

测试

当正常出现版本号，就说明已经安装成功了
